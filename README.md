# Fullboard Guest Meal Management System

I practice native PHP first and setup manual MVC and this project i use laravel now! I ask my partner whats the problem in his work on resort and i built this to solve the problem by managing guest who avail fullboard meal to records
(breakfast, lunch, dinner) to prevent repeating meals if the guest visit again after a weeks or a day, or even a month, and also with room assignment and checkout tracking.

## Features
- Guest check-in & checkout
- Record guest meals
- Daily meal tracking
- Room assignment
- Displaying Available Fullboard Meal for Today
  

## Technologies Used
- Laravel
- Blade
- MySQL
- Tailwind CSS
